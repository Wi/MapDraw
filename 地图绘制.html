地图绘制


1、坐标转换


扫地机的坐标系采用世界坐标，所以返回的数据点需要根据乐动地图相关文档定义转成地图坐标系，地图坐标系 App 并不能直接使用，还需转成手机像素坐标系。

世界坐标系->地图坐标系->像素坐标系

A、世界坐标系->地图坐标系


世界坐标系转成地图坐标系算法相对复杂，以下是详细实现技术方案。

1）、设备返回世界坐标的距离单位为米，首先要转成厘米计算；

2）、减去世界坐标xy值的偏移量（xmin/ymin)；

3）、世界坐标转地图坐标分辨率转换，x、y 值除以地图步长(resolution)。

/* 转换代码
@param x_min 地图左下角对应的x世界坐标(米)
@param y_min 地图左下角对应的y世界坐标(米)
@param resolution 分辨率:表示每个像素代表的世界坐标系下的距离，单位米
 */
CGPoint mapPoint = CGPointZero;
    
mapPoint.x = (worldPoint.x / 1000 - x_min) / resolution;
mapPoint.y = (worldPoint.y / 1000 - y_min) / resolution;
    
return mapPoint;


B、地图坐标系->像素坐标系


如上图中所示，可见地图坐标系与像素坐标中的 y 坐标是反转的，所以需要针对 y 值做一次翻转。

/* 关键代码如下 */
    
CGPoint pixPoint = CGPointZero;
pixPoint.x = mapPoint.x;
pixPoint.y = size.height - mapPoint.y;
return pixPoint;


2、地图绘制


A、地图绘制


首先将地图数据解析出来的一维数组根据 mapSize 转成二维数组，形成平面图的基础数据源，按不同的种类将地图点的类型分为以下三类：

typedef enum : NSInteger{
    MapPathType_Obstacle = 0,   // 障碍物
    MapPathType_Unknow = 127,   // 未知区域
    MapPathType_Blank = 255     // 空白（安全区域）
}MapPathType;

同时，生成三条贝塞尔曲线，遍历数据源，将转换成像素坐标后的地图点，根据不同的类型，分别添加进贝塞尔曲线中，形成三条闭合的贝塞尔曲线。对贝塞尔曲线进行颜色填充，子线程异步绘制成图片。到这里，已经完成了基础的图形绘制。


B、路径绘制


路径绘制的原理跟地图绘制的原理一样，这里不再赘述。区别在于路径的绘制针对链接处做了平滑处理，这样会使路径看起来更加流畅、连贯。


3、地图优化


A、地图降噪、墙边拉直

地图降噪，是把地图中散乱且无用的障碍点去除的过程，优化后使地图更加美观、整洁。
墙边拉直、墙边细化，是指在不对地图轮廓和障碍物造成影响的前提下，对墙边进行拉直及细化的过程。

以下是处理前后的对比图：

地图优化初期，了解了图像边缘处理、边缘细化相关算法，如虫随法、注水算法等，并还原实现了相关算法，但预期效果不佳，不能满足项目需求。最终通过仔细观察效果图，分析地图数据，并做了大量的实践得出比较完善的优化策略。
图形处理的基本原理，是将一个点与其相邻八个点组合成一个矩阵，以矩阵为最小计算单位（分子），如下：


    P9    
    P2    
    P3    

    P8    
    P1    
    P4    

    P7    
    P6    
    P5    

分析其与周围点之间的连通性，最后通过计算相邻各个点位的连通总和来判断该点是否需要被清除。
遍历数据中的点，将不连通的点标记为0，连通的点标记为1.求出 p2-p9的连通总和(SUM)。
SUM = p2 + p3 + p4 + p5 + p6 + p7 + p8 + p9, 如：


     0     
    0    
     1     

    1    
    P1    
    0    

    1    
    0    
    1    

SUM = 3;


     0     
    0    
     0     

    0    
    P1    
    0    

    0    
    0    
    0    

SUM = 0;

如果为0，则表示该点不与其他点相连，可以被清除；否则说明该点与其他点相连，会影响图形轮廓结构，不能被清除。部分关键代码如下：

    /* 部分关键代码 */
    int flag = 1;
    int p1 = 0, p2 = 0, p3 = 0;
    int p4 = 0, p5 = 0, p6 = 0;
    int p7 = 0, p8 = 0, p9 = 0;
    int offset = 0;
    for (int row = 1; row < height - 1; row ++)
    {
        offset = row * width;
        for (NSInteger col = 1; col < width - 1; col ++)
        {
            p1 = [data[offset + col] intValue];
            p2 = [data[offset - width + col] intValue];
            p3 = [data[offset - width + col + 1] intValue];
            p4 = [data[offset + col + 1] intValue];
            p5 = [data[offset + width + col + 1] intValue];
            p6 = [data[offset + width + col] intValue];
            p7 = [data[offset + width + col - 1] intValue];
            p8 = [data[offset + col - 1] intValue];
            p9 = [data[offset - width + col - 1] intValue];
            
            if (p1 != flag) {
                continue;
            }
            
            int sum = p2 + p3 + p4 + p5 + p6 + p7 + p8 + p9;
            if (sum == flag)
            {
                [handledArray replaceObjectAtIndex:offset + col withObject:@(!flag)];
            }
        }
    }
